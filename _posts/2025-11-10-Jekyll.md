## 数学公式渲染
如果主题为：`jekyll-theme-chirpy`，且 `jekyll` 版本为 `4.4.1`（终端命令 `jekyll -v` ）。

在 `_config.yml` 和笔记页面中增加`math: true`。具体如下所示

```
_config.yml

# The Site Configuration

# Import the theme
theme: jekyll-theme-chirpy
math: true
```

```
笔记页面顶部

---
title: Jekyll使用
date: 2025-11-10 08:00:00 +0800 
categories: [机器学习] 
tags: [工具]
math: true
---
```

行内公式中输入`|`，形如：`$xxx|xx$`会存在渲染问题。

解决办法：
1. $$a|_{a=1}$$
   其中，`|`可以替换为`\left. xxx \right|_{}`、`\big|`
2. $a\vert _{a=1}$
